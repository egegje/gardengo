# SemenovLab — лендинг

Статический сайт студии фитодизайна SemenovLab + встроенный визуальный редактор.

- **Сайт:** `index.html` (публичная страница)
- **Редактор:** `admin.html` (правка любого текста и фото прямо на странице, публикация в этот репозиторий)
- **Контент:** `content.json` (единый источник всех текстов и картинок)
- `app.js` — рендер, `styles.css` — стили

Реализовано из дизайн-хэндоффа Claude Design (`SemenovLab.dc.html`).
